Руководство к использованию программы
В файл in.txt заносятся таблицы истинности (под номерами, которые в дальнейшем используются для выведения результата, номера – натуральные числа; построчно: одна таблица истинности – одна строка). Приведем пример одной из таких строк:
1	01010101
где 1 – порядковый номер, а 01010101 – таблица истинности (то есть, значения отклика, соответствующие значениям уровней факторов, отсортированных по возрастанию).
Далее производится запуск программы. Время работы программы критически зависит от входных значений (чем больше размерность булева куба – тем дольше будет работать программа).
По окончании работы результат будет записан в файл out.txt в формате
<спектр> = [номера таблиц истинности соответствующих откликов из файла in.txt через запятую]
